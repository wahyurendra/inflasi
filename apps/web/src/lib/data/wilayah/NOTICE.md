# Wilayah data attribution

`regencies.json` (kabupaten/kota) and `districts.json` (kecamatan) are derived
from [idn-area-data](https://github.com/fityannugroho/idn-area-data)
(`regencies.csv`, `districts.csv`, v4.0.1), licensed under the
[Open Database License (ODbL)](https://opendatacommons.org/licenses/odbl/).

Converted to JSON and trimmed to the province/regency/district levels this
app needs (village/kelurahan-level data is not included).
